# 🎥 直播复盘仪表盘

一个用于展示和分析直播数据的可视化仪表盘网页。

## ✨ 功能特性

- 📊 **战役总览** - 展示所有场次的宏观走势
- 📈 **趋势图表** - 支持切换查看成交单数、场观人数、转化率
- 📅 **场次选择** - 横向滚动的时间轴选项卡
- 📋 **详细复盘** - 每场直播的数据漏斗和文字复盘

## 🛠️ 技术栈

- 原生 HTML5 / CSS3 / JavaScript
- Chart.js - 图表库

## 🚀 使用方法

直接用浏览器打开 `index.html` 即可使用。

## 📁 项目结构

```
├── index.html          # 主页面文件
├── README.md          # 项目说明
├── 需求.md            # 原始需求文档
├── 直播复盘数据.txt    # 原始数据
├── 1月31日直播复盘.pdf # 历史复盘
└── 2月5日直播复盘.pdf  # 历史复盘
```

## 📊 数据包含

4场直播的完整数据：
- 1月31日 - 春节攻势·启动战
- 2月5日 - 口碑发酵·升温战
- 2月8日 - 节前冲刺战
- 2月12日 - 年前收官战
